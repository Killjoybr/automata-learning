# Aprendizado de Autômatos e Linguagens Formais

## Sobre este Repositório

Este repositório foi criado para armazenar minhas anotações sobre o aprendizado de autômatos e linguagens formais, juntamente com alguns exercícios práticos.

## Estrutura do Repositório

O repositório está organizado nas seguintes pastas:

- **Gramática**: Anotações e exercícios sobre gramáticas formais
- **Autômatos**: Conteúdo relacionado a autômatos finitos e suas aplicações
- **Máquina de Turing**: Estudos sobre máquinas de Turing e computabilidade

## Objetivo

O objetivo deste repositório é documentar minha jornada de aprendizado em teoria da computação, servindo como material de referência e prática.
